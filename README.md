# LeetCode Practice Notebook

A Jupyter notebook for practicing LeetCode problems.

## Getting Started

1. Clone this repository
2. Open `leetcode_practice.ipynb` in Jupyter Notebook or JupyterLab
3. Use the provided templates to solve LeetCode problems

## Features

- Problem template with sections for problem details and solution
- Test case template for verifying solutions
- Example solution (Two Sum) to demonstrate usage

## Requirements

- Python 3.x
- Jupyter Notebook or JupyterLab

## Usage

1. Copy the problem template cells for each new problem
2. Fill in the problem number, title, difficulty, link, and description
3. Implement your solution in the Solution class
4. Add test cases and run them to verify your solution
